Predicting Ages of Abalone




Summary of Project

Our data set was created by the UC Irvine Machine Learning Repository and lists various physical characteristics present in the abalone samples. Several characteristics include age, sex, and weight, but the most important trait for this project was the number of rings within the shell. The larger the number of rings, the more mature the abalone was, and would therefore be a sign of when to harvest it. After entering, transforming, and loading the data, we began testing our data and were attempting to obtain an acceptable score for our model, yet we found that the data is not efficient enogh to allow us to make a prediction.

Data Set

UC Irvine Machine Learning Abalone Data Set

Tools

Google Colabs was our main program of choice to run our Multiple Linear Regression tests as it was the most suited for our tests.

Models

The main model that we used to do our test was a Multiple Linear Regression as there were multiple variables that we wanted to see the relationship between with the ring count in the abalone shells. After running the model, we obtained a score of 0.5459, which was average, but not acceptable for our test. In an attempt to get cleaner results, we ran multiple other tests, including K Nearest Neighbors, Support Vector Regression, and Random Forest Regression, but none of them returned better results.
